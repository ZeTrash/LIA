# Livrables Étape 2.6

Ce dossier regroupe les descriptions formelles promises pour le système d'autonomie et boucle autonome.

| Fichier | Description |
| --- | --- |
| `specification_scheduler.md` | Spécification complète de `LIAAutonomousScheduler` (interface, boucle principale, intervalles, gestion erreurs). |
| `specification_objectifs_personnels.md` | Extension memory_service pour objectifs personnels (modèle données, API, conditions déclenchement). |
| `specification_portails.md` | Spécification des trois portails (Autonome, Multi-Agent, Humain) avec interfaces et algorithmes. |
| `algorithmes_metriques_autonomie.md` | Algorithmes de calcul des métriques d'autonomie (taux tromperie, indice autonomie, exploration, stabilité). |
| `architecture_technique.md` | Architecture technique détaillée, composants, flux d'exécution, performance, déploiement. |
| `plan_tests_validation.md` | Plan de tests automatisés, scénarios de validation et critères d'acceptation. |

Ces artefacts reprennent fidèlement le périmètre SL1→SL5 décrit dans `README.md` et peuvent être validés indépendamment.
