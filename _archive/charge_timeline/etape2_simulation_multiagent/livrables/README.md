# Livrables Étape 2

Ce dossier regroupe les descriptions formelles promises pour le service de simulation multi-agent.

| Fichier | Description |
| --- | --- |
| `protocol_message_schema.json` | JSON Schema pour les messages échangés entre agents. |
| `protocol_handshake_schema.json` | JSON Schema pour le protocole de handshake initial. |
| `api_spec_simulation.yaml` | Spécification OpenAPI 3.1 complète du service de simulation. |
| `algorithmes_metriques.md` | Algorithmes détaillés de calcul des métriques comportementales (variabilité, autonomie, curiosité, cohérence). |
| `plan_tests_validation.md` | Plan de tests automatisés, scénarios de validation et critères d'acceptation. |
| `architecture_technique.md` | Architecture technique, composants, flux d'exécution et gestion des erreurs. |

Ces artefacts reprennent fidèlement le périmètre SL1→SL4 décrit dans `README.md` et peuvent être validés indépendamment.
